How to split a PDF

Add your PDF, then choose what you want to do. Use Extract pages to pull out specific pages or ranges — type something like 1-3, 5, 8-10 — and get a single new PDF with just those pages. Use Split into files to break the document into chunks of a set number of pages; set it to 1 to separate every page into its own file. The result downloads instantly, with multiple files delivered together as a ZIP.

Separate PDF pages privately

Unlike most online splitters that upload your file to a server, this tool does everything on your device using JavaScript. Your PDF — contracts, statements, scans — never leaves your computer, which is both more private and faster.

Frequently asked questions

How do I extract pages from a PDF?

Choose Extract pages, type the page numbers or ranges you want (for example 1-3, 5), and click Extract. A new PDF containing only those pages downloads right away.

How do I split a PDF into multiple files?

Choose Split into files, set how many pages each file should contain (1 for one page per file), and click Split PDF. The resulting files download together in a ZIP.

How do I separate pages in a PDF document?

Set "Split every" to 1 — each page becomes its own PDF, so every page is fully separated.

Is my PDF uploaded anywhere?

No. Splitting happens entirely in your browser, so your document is never uploaded — ideal for sensitive files.

Will the split files have a watermark?

No. The output PDFs are clean, with no watermark and no quality loss.
