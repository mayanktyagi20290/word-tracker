FAQ

How many URLs can I add?
Up to 50,000. If your list exceeds the split threshold (default 1,000 URLs — Google's recommendation), a sitemap index file is automatically generated alongside numbered child sitemaps. Upload all of them to your root directory, then submit the index URL to Google Search Console.
Can I set different settings for /blog/ vs /products/?
Yes — that's what folder rules are for. Add a rule for /blog/ with weekly frequency and 0.7 priority, and another for /products/ with daily and 0.9 priority. Each URL is matched against rules top-to-bottom and the last match wins.
Do I need to crawl my site?
No. Paste your URL list directly — export it from Google Search Console, Screaming Frog, Ahrefs, or your CMS. This avoids crawl timeouts, bot-protection blocks, and rate limits entirely.
What is a sitemap index and when is it created?
A sitemap index is a master XML file that links to child sitemaps. This tool creates it automatically when your URL count exceeds your chosen split threshold. Submit only the index URL to Google — it discovers the child files on its own.
