Frequently asked questions

How many URLs can I add to this sitemap generator?
Up to 50,000 URLs. If your list exceeds the split threshold (default: 1,000 URLs), the tool automatically generates a sitemap index file linking to multiple child sitemaps, which is Google's recommended approach for large sites.
Can I set different priority and changefreq for different folders?
Yes. The Folder Rules panel lets you add path patterns like /blog/ or /products/ and assign custom priority, changefreq, and lastmod to every URL matching that path. Rules are applied top-to-bottom so more specific paths can override general ones.
What is a sitemap index file and when is it generated?
A sitemap index is a master XML file that points to multiple child sitemap files. This tool auto-generates both the index and child files when your URL count exceeds your chosen split threshold. You upload all files to your root directory, then submit just the index URL to Google Search Console.
Does this tool crawl my website?
No. You paste your list of URLs directly. This makes it work instantly for any number of URLs without crawling limits or bot-protection issues. Export your URL list from Google Search Console, Screaming Frog, or your CMS and paste it here.
What format should I paste my URLs in?
One URL per line. Full URLs (https://example.com/page) or relative paths (/page) both work — relative paths are resolved using the base domain you enter. Duplicates and blank lines are removed automatically.
